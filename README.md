# OpenWeather App

## Project Description

Its a minimal App that displays the Weather in location (Cities) across the World. App was developed using Node.js (Express)
Advanced version will come soon.
  
## Build With

  * HTML
  * CSS
  * JavaScript
  * Node.js
  * Express
